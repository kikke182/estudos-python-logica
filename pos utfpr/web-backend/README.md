#  Desenvolvimento Web Backend com Flask

Este repositório é o meu acervo pessoal de consultas, códigos e anotações. O objetivo é registrar o progresso das aulas semana a semana e documentar os principais conceitos de arquitetura web, APIs e persistência de dados.

##  Cronograma e Índice de Estudos

* **[Semana 1 - Flask e Python para Back-End](./semana1):** Configuração do ambiente, introdução ao ecossistema do Flask e conceitos fundamentais de servidores web.
* **[Semana 2 - Flask e o HTTP](./semana2):** Anatomia das requisições e respostas HTTP (Headers, Verbos/Métodos) e organização modular de rotas utilizando **Blueprints**.
* **[Semana 3 - Flask e Conteúdos Estáticos e Dinâmicos](./semana3):** Renderização de templates, arquivos estáticos (CSS/JS) e integração básica de visões dinâmicas.
* **[Semana 4 - Flask e o SQLite](./semana4):** Integração com banco de dados relacional, ciclo de vida de conexões (`g` do Flask, context teardown) e operações CRUD básicas seguros contra SQL Injection.
* **[Semana 5 - Flask e Parâmetros do HTTP](./semana5):** Captura de dados dinâmicos via URL Parameters, Query Strings (`?key=value`) e dados de formulários/JSON enviados no corpo da requisição.
* **[Semana 6 - Flask e Cookies/Sessions/JWT](./semana6):** Mecanismos de persistência de estado no protocolo HTTP (stateless). Diferenças práticas e implementação de Cookies, Sessões de Servidor e Tokens JWT para autenticação.
* **[Semana 7 - Flask e MISC](./semana7):** Tópicos diversos, utilitários, refatorações e organização de código avançada.
* **[Semana 8 - Flask e Segurança](./semana8):** Boas práticas de segurança em aplicações web baseadas nas diretrizes do OWASP (Open Web Application Security Project).

---
*Nota: A estrutura conceitual, os desafios e os códigos base foram desenvolvidos a partir do material didático disponibilizado pelo professor.
